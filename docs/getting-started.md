# Getting Started

This guide will help you navigate and make the most of this nexus.

## Navigation

### Main Menu

The navigation bar at the top provides quick access to:

- **Guide** - Main documentation of how this nexus is structured.
- **Projects** - Dropdown menu with all available project documentation
- **Blog** - Latest updates and articles
- **GitHub** - Source code repositories

### Sidebar Navigation

Each documentation section has a sidebar on the left that shows the relevant pages and their childrens. The pages will be organized hierarchially.

### Search

Use the search bar (keyboard shortcut: `Ctrl+K` or `Cmd+K`) to quickly find:
- Specific topics
- Code examples
- Configuration details
- Any content across all documentation

## Dark Mode

Toggle between light and dark themes using the switch in the navigation bar. Your preference is automatically saved.

## Version Selection

For projects with multiple versions:
1. Navigate to the project documentation
2. Use the version dropdown in the top-right corner
3. Select the version you need

The latest stable version is selected by default.

## Mobile Navigation

On mobile devices:
- Tap the hamburger menu (☰) to access navigation
- Swipe from the left edge to open the sidebar
- All features are fully responsive

## Reading Tips

- **Code blocks** can be copied with the copy button in the top-right corner
- **Links** are highlighted and clickable throughout the nexus
- **Headings** have anchor links for easy sharing
- **Tables of contents** appear on the right for longer pages
