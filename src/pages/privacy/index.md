# Privacy Policy

Hi there. This is a personal site for portfolio and documentation of my projects. By using this site, you agree to the following:

### No Tracking
I don't use any invasive tracking cookies or analytics that follow you around the web. I'm not interested in your personal data, and I don't sell it to anyone.

### What I Collect
Technically, nothing. This site is static. If you email me or reach out via GitHub, I'll have whatever information you choose to share in that message. That's it.

### Third Parties
This site is hosted on Vercel (or similar). They might collect some basic server logs for security and performance (like IP addresses), which is standard for how the internet works.

### Changes
If I ever decide to change this policy, I'll update this page. But honestly, I plan to keep it this simple.
