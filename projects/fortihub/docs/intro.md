# Introduction

FortiHub is something I built to solve a simple problem for personal projects. I needed a proper auth system for multiple projects and I didn't want to do the same thing again and again. So, I just built this. This will act as a microservice to manage my authentication flow and I won't even have to go through the hassle of setting up auth for each project. However, even though I am using it, it isn't perfect and has a few limitations, among which the inability to constraint the user table from another project ranks first.
