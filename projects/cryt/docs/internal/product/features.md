# Core Features

## Messaging

* Real-time text chat
* Data-channel based transport
* No server-side message storage

## Ephemeral Rooms

* Rooms are created instantly
* Rooms expire automatically after inactivity
* No user accounts required

## Encoding Modes

* Morse code (dot–dash visualization)
* Caesar cipher
* Base64
* User-defined substitution cipher

## Security Modes

* Plaintext
* Encoded-only
* End-to-end encrypted (AES-GCM)

---

# Non-Goals

Cryt is **not** intended to be:

* a social network
* a persistent messaging platform
* a file-sharing service

The focus is:

> temporary, experimental, and private communication.
