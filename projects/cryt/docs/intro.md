# Introduction

**Cryt** (formerly DotDash) — *Ephemeral peer chat with playful encryption.*

> A browser-based ephemeral chat that lets users communicate using Morse code, custom ciphers, and end-to-end encryption — with no message storage and no permanent accounts.

## Problem Statement

Most modern chat platforms:

* store messages permanently
* require accounts or phone numbers
* lack flexibility in how messages are encoded or presented

At the same time, existing encrypted chat apps focus on security but not on **creativity, experimentation, or learning**.

There is no simple tool that allows users to:

* communicate privately
* experiment with encoding or cryptography
* create temporary conversations that disappear automatically

## Solution

**Cryt** is a web application that provides:

* ephemeral chat rooms
* real-time communication
* customizable encoding layers (Morse, Caesar, Base64, custom)
* optional end-to-end encryption
* zero message persistence

Users can:

* share a room link
* chat in real time
* encode and decode messages in creative ways
* leave no trace after the session ends

## Target Use Cases

Cryt is intended for:

* privacy-conscious casual conversations
* teaching Morse code or classical cryptography
* experimenting with custom encodings
* temporary collaboration sessions
